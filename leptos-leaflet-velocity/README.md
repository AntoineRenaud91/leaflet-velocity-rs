# leptos-leaflet-velocity

A [Leptos](https://leptos.dev/) component for [leaflet-velocity.js](https://github.com/onaci/leaflet-velocity), providing animated velocity/wind visualizations on Leaflet maps.

## Installation

```toml
[dependencies]
leptos-leaflet-velocity = "0.3"
```
